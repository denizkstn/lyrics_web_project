## About The Project
This repository contains the source code for a static web application dedicated to showcasing a curated collection of 15 songs. Built entirely with core web technologies (HTML5 and CSS3), the platform provides users with clean and accessible track pages that include full lyrics, producer credits, and detailed song metadata.

This project was developed as an academic assignment to master the fundamentals of web design, semantic HTML structuring, and CSS styling without relying on external frameworks.

## Key Features
* **Detailed Song Directory:** Contains a categorized library of 15 specific tracks.
* **Rich Metadata:** Displays comprehensive information for each song, including lyrics and producer details.
* **Semantic Structure:** Built with clean, readable, and well-structured HTML5.
* **Custom Styling:** Fully styled using pure CSS for a straightforward and user-friendly interface.

## Built With
* **Languages:** HTML5, CSS3
* **Environment:** macOS / Unix, VS Code

## Project Status & Context
*This project was built as part of my academic curriculum at Izmir University of Economics. The primary goal was to solidify my understanding of core front-end web development principles (DOM structure, styling, and static content delivery) before moving on to more complex JavaScript frameworks.*
